# Pet Store Service
This repository contains backend codes of Pet Store. All codes written in Java.

# How do I get set up?
-Clone this repo

-Install Java

-Instal eclipse or Spring Tool Suite

-Import Project 

-Run as Spring Boot App

1.SpringBoot, as framework for backend services
